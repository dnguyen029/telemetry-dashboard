---
description: Perform systematic root-cause analysis and resolve runtime issues or bugs
---

When the user types `/debug <error>`, coordinate the agents as follows:

1. Act as the **orchestrator** to evaluate symptoms, determine technical root cause, and draft a Root Cause Analysis (RCA).
2. Act as the **builder** to implement the permanent fix without patching symptoms.
3. Act as the **admin** to verify that standard test runs execute successfully.
4. Act as the **auditor** to verify that the fix does not introduce regression or safety gaps.
