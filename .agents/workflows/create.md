When the user types `/create <requirements>`, coordinate the agents as follows:

1. Act as the **orchestrator** to define files, components, and layout boundaries based on the `<requirements>`.
2. Act as the **builder** to scaffold code structures, initialize packages, and write implementation logic.
3. Act as the **auditor** to verify code boundaries, linting, and proper safety practices are satisfied.
