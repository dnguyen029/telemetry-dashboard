# 🔍 Review Master Skill

Unified authority for performing high-precision code reviews and PR enhancements.

## 🎯 Review Standards

- **Correctness**: Verify logic against mission goals.
- **Security**: Audit for leaks, XSS, and insecure dependencies.
- **Performance**: Check for O(n^2) loops and memory leaks.
- **Style**: Enforce project DNA (ESM, linting).

## 🛠️ Workflow

1. Run static analysis/linting.
2. Verify test coverage.
3. Perform manual logic walkthrough.
4. Generate structured feedback (Zod-ready).
